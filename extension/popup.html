自定义模型接口
这里只负责保存模型接口和 GitHub Token。当前版本默认按 OpenAI 兼容的 `chat/completions` 接口调用。
查看模型配置说明与示例
模型接口地址怎么填
建议填写兼容 OpenAI 的 `/v1` 根地址，例如 `https://api.openai.com/v1` 或 `https://api.example.com/v1`。如果服务商给的是完整 `/chat/completions` 地址，也可以直接填写。
模型名称怎么填
填服务商后台显示的真实模型名，例如 `gpt-4o-mini`、`gemini-3.1-flash-lite`。名称需要和平台支持的模型完全一致。
GitHub Token 的作用
公开仓库场景可以先不填，但加上 Token 后，读取 GitHub 仓库信息会更稳定，也更不容易遇到限流。
扩展会自动补成 `chat/completions` 调用地址，所以大多数平台直接填写 `/v1` 即可。
配置保存在当前浏览器扩展的本地存储中，请根据所在环境的安全要求管理密钥。
